PyParsers
#########
PyParsers is a pure python3 micro framework that aims at letting you easily design and implement a parser for the DSL you have in mind. Concretely, it pursues the following two objectives:

You shouldn't need to learn yet another framework for doing what you want.
The code should be easily understood, even in a far away future.
